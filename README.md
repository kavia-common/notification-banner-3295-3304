# notification-banner-3295-3304

This workspace contains the React frontend for a simple notification demo that uses Tailwind CSS and an Ocean Professional theme.

## Containers
- notification_frontend: React app that shows a top-right notification which auto-dismisses after 3 seconds.

## Development
- npm start to run locally (from notification_frontend directory)
- npm test to run tests

See notification_frontend/README.md for details.
